# Challenge ingreso a Academia ForIT 2025

## Ejecutar aplicasion

Luego de clonar este repositorio, nos hubicaremos en la carpeta desde la consola y ejecutaremos:

```bash
npm install
```

esto instalara las dependencias del proyecto.

Luego ejecutaremos, en el mismo lugar:

```bash
npm run dev
```

Esto levantara el proyecto (tanto front-end como el back-end )
